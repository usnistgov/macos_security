Running the Compliance Script

Note

The script must run in zsh (default on macOS). Using bash or sh will cause errors.

Interactive mode:

Automated mode:

FlagWhat it does
--checkRun checks only
--fixRun fixes only
--cfcCheck → Fix → Check
--statsShow last run statistics
--compliantReport compliant count
--non_compliantReport non-compliant count
--resetClear results for this baseline
--reset-allClear results for all baselines
--quiet=1Show failed/exempt only

Always Use the Correct Branch

The main branch is now used for mSCP 2.0 and will generate content using the mSCP 2.0 methods. For mSCP 1.0, each macOS version has its own branch (sequoia, sonoma, ventura, etc.) — check out the branch that matches your target OS.

Quick Start

Requirements:

  • Python >= 3.12.1 (3.14 is not supported)
    • Recommended: Macadmins Python
  • Ruby (macOS built-in Ruby works — optional, for PDF output)

  1. Clone & Checkout Branch

    Replace sequoia with your target macOS version (sequoia, sonoma, ventura, etc.).

  2. Install Dependencies

    Ruby (optional — for PDF output):

  3. Generate a Baseline

    List baselines: ./scripts/generate_baseline.py -l

    Generate: ./scripts/generate_baseline.py -k BASELINE_NAME

    With tailoring: ./scripts/generate_baseline.py -k BASELINE_NAME -t

  4. Generate Outputs

    Run generate_guidance.py with the flags you need:

    FlagOutput
    (none)Guidance docs (.adoc, .html, .pdf)
    -sCompliance script
    -pConfiguration profiles (one per payload)
    -PSingle consolidated profile
    -DDDM components
    -xExcel spreadsheet
