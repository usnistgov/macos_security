## Key Files

<Tabs syncKey="mscp-version">
  <TabItem label="mSCP 2.0">


**800-53_baselines.yaml** тАФ Contains the NIST 800-53 control mappings used by the `-c` flag in `./mscp.py baseline` to output controls covered by rules.

**mscp-data.yaml** тАФ Project-wide metadata including version information, supported platforms, and default settings.

**acronyms.yaml** тАФ Acronym definitions used in generated documentation.

  </TabItem>
  <TabItem label="mSCP 1.0">

<Aside type="caution" title="Always Use the Correct Branch">
The `main` branch is now used for **mSCP 2.0** and will generate content using the mSCP 2.0 methods. For mSCP 1.0, each macOS version has its own branch (`sequoia`, `sonoma`, `ventura`, etc.) тАФ check out the branch that matches your target OS.
</Aside>


**supported_payloads.yaml** тАФ Defines which configuration profile payloads the project supports. When adding custom rules that use new payloads, add them here.

**800-53_baselines.yaml** тАФ Contains the NIST 800-53 control mappings used by the `-c` flag in `generate_baseline.py` to output controls covered by rules.

**mscp-data.yaml** тАФ Project-wide metadata including version information and default settings.

  </TabItem>
</Tabs>
