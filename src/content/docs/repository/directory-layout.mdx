## Repository Structure

<Tabs syncKey="mscp-version">
  <TabItem label="mSCP 2.0 (Beta)">

<FileTree>
- config/
    - default/
        - rules/ Individual rule YAML files
        - sections/ Section definition files
        - templates/ Documentation templates
        - images/ Project images
        - themes/ Theme files
    - custom/
        - rules/ Custom rule overrides
        - baselines/ Generated/tailored baselines
        - sections/ Custom section overrides
        - templates/ Custom templates
        - images/ Custom images
    - includes/ Supporting YAML files and resources
    - locales/ Localization files
- src/
    - mscp/ Python CLI package
- schema/ YAML schema definitions
- build/ Generated output files
- rules → config/default/rules (symlink)
- baselines → config/default/baselines (symlink)
- custom → config/custom (symlink)
</FileTree>

  </TabItem>
  <TabItem label="mSCP 1.0">

<FileTree>
- baselines/ Baseline YAML files for compliance frameworks
- build/ Generated output files
- custom/ Organization-specific customizations
- includes/ Supporting YAML libraries and resources
- rules/ Individual rule YAML files
- scripts/ Generation and utility scripts
- sections/ Section definition files for guides
- templates/ AsciiDoc templates for documentation
</FileTree>

  </TabItem>
</Tabs>

## Rules Directory

The rules directory contains subdirectories organized by category:

<Tabs syncKey="mscp-version">
  <TabItem label="mSCP 2.0 (Beta)">

| Category | Description |
|:---------|:------------|
| `audit/` | OpenBSM auditing settings |
| `auth/` | Authentication and smartcard settings |
| `icloud/` | Apple iCloud and Apple ID configuration |
| `os/` | General operating system settings |
| `pwpolicy/` | Password policy enforcement |
| `settings/` | Application and service settings |
| `system_settings/` | System Settings configuration |

  </TabItem>
  <TabItem label="mSCP 1.0">

| Category | Description |
|:---------|:------------|
| `audit/` | OpenBSM auditing settings |
| `auth/` | Authentication and smartcard settings |
| `icloud/` | Apple iCloud and Apple ID configuration |
| `os/` | General operating system settings |
| `pwpolicy/` | Password policy enforcement |
| `supplemental/` | Additional supporting information |
| `system_settings/` | System Settings configuration |

  </TabItem>
</Tabs>
